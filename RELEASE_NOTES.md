# AIMED Release Notes

## v0.1.0 - Initial Release (2025-10-09)

Initial public release of AIMED (AI Monitoring & Execution Dashboard) - a web interface for ConPort MCP providing visual access to AI project context management.

### What's Included

- Web dashboard with activity feed, Kanban board, and knowledge graph visualization
- Full-text and semantic search across all project context data
- Visual editors for product and active contexts
- CRUD interfaces for decisions, progress, patterns, and custom data
- Multi-workspace support with automatic isolation
